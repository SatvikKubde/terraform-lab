# Terraform Infrastructure as Code Lab

## Objective
This lab demonstrates Infrastructure as Code (IaC) using Terraform.

## Tools Used
- Terraform
- Docker
- Git & GitHub
- GitHub Actions

---

## Assignments Overview

### Day 1: Introduction to Terraform
- Created a file using local provider
- Used variables and outputs

### Day 2: Variables and Environments
- Used dev.tfvars and prod.tfvars
- Created environment-specific configurations

### Day 3: Docker with Terraform
- Pulled nginx image
- Created container on port 8080

### Day 4: Terraform Modules
- Created reusable module
- Deployed multiple containers

### Day 5: CI/CD Automation
- Implemented GitHub Actions workflow
- Automated terraform fmt, validate, and plan

### Bonus: Dynamic Infrastructure
- Used for_each to create multiple containers dynamically

---

## Learning Outcomes
- Terraform workflow (init, plan, apply)
- Infrastructure automation
- Modular design
- CI/CD integration
- Dynamic resource creation
